## 📌 Project Overview
This project is an interactive **Power BI Business Intelligence Dashboard** developed for **Sales Performnace Analysis**.
The dashboard helps management monitor business performance through key metrics such as sales, profits, regional performance, category achievements, delivery risks, and return analysis. It enables data-driven decision-making using dynamic visualizations and interactive filters.

---

## 🚀 Features

- Executive Business Overview
- Sales Performance Analysis
- Region-wise Performance Tracking
- Manager Performance Comparison
- Category Target Achievement
- Return Analysis
- Delivery Risk Monitoring
- Interactive Slicers & Filters
- KPI Cards
- Dynamic Charts and Graphs

---

## 📑 Dashboard Pages

### 1️⃣ Executive Overview
Provides a high-level summary of the business including:

- Total Sales
- Total Profit
- Orders
- KPIs
- Overall Business Performance
- Trend Analysis

---

### 2️⃣ Region & Manager Performance

Analyzes business performance based on:

- Region-wise Sales
- Manager-wise Performance
- Regional Comparison
- Revenue Contribution
- Performance Distribution

---

### 3️⃣ Category Target Achievement

Tracks performance against business targets.

Includes:

- Category-wise Sales
- Target vs Achievement
- Performance Comparison
- Goal Completion Analysis

---

### 4️⃣ Return & Delivery Risk

Focuses on operational efficiency by analyzing:

- Product Returns
- Delivery Performance
- Risk Identification
- Return Trends
- Delivery Issues
